# Septic Shock Vasopressor and Hemodynamic Management Protocol

## Septic Shock Recognition

**Septic shock** = sepsis + ALL of the following despite adequate fluid resuscitation:
- Persistent MAP <65 mmHg requiring vasopressors, AND
- Serum lactate ≥2 mmol/L

**Immediate Actions on Recognition:**
1. Call attending physician and ICU team **NOW**
2. Place arterial line for continuous BP monitoring
3. Ensure large-bore IV access (≥2 peripheral or central line)
4. Initiate vasopressors — do not wait for central line placement

## Vasopressor Selection and Dosing

### First-Line: Norepinephrine (NE)
- **Start:** 0.01–0.05 mcg/kg/min IV
- **Target:** MAP ≥65 mmHg
- **Titrate:** Increase by 0.02–0.05 mcg/kg/min every 5–10 minutes
- **Max:** 0.5–1.0 mcg/kg/min before considering adjuncts

### Second-Line: Vasopressin (add to NE, do not titrate)
- **Dose:** Fixed 0.03 units/min IV
- **Indication:** NE dose ≥0.25 mcg/kg/min and MAP still <65 mmHg
- **Benefit:** Reduces NE requirements and may improve renal perfusion

### Third-Line: Epinephrine
- **Start:** 0.01–0.05 mcg/kg/min
- **Indication:** MAP still <65 mmHg on NE + vasopressin
- **Caution:** Increases lactate — do not use lactate clearance as sole resuscitation endpoint when on epinephrine

### NOT Recommended
- Dopamine as vasopressor for septic shock (increased arrhythmia risk vs NE)
- Phenylephrine as first-line (reduces cardiac output)

## Fluid Resuscitation

- **Initial bolus:** 500 mL crystalloid over 15–30 minutes; reassess
- **30 mL/kg rule:** Administer up to 30 mL/kg total for hypoperfusion in first 3 hours
- **Stop resuscitation if:** SpO2 decreasing, new pulmonary crackles, CVP rising rapidly
- **Prefer:** Balanced crystalloids (lactated Ringer's, Plasmalyte) over normal saline

## Hemodynamic Targets

| Parameter | Target |
|---|---|
| MAP | ≥65 mmHg (≥80 if prior hypertension) |
| Heart rate | <110 bpm (after resuscitation) |
| Urine output | ≥0.5 mL/kg/hour |
| Lactate clearance | ≥10% reduction per 2 hours |
| ScvO2 | ≥70% |

## Refractory Shock Escalation

If MAP <65 mmHg persists despite NE ≥0.5 mcg/kg/min:
1. Add vasopressin 0.03 units/min if not already started
2. Add epinephrine 0.01–0.05 mcg/kg/min
3. Consider IV hydrocortisone 200 mg/day (relative adrenal insufficiency)
4. Evaluate for reversible cause: tension pneumothorax, cardiac tamponade, massive PE
5. Activate rapid response / code team if not already done
6. Consider ECMO consultation for refractory cases at ECMO-capable center
