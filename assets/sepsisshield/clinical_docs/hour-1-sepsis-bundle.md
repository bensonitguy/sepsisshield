# Hour-1 Sepsis Bundle

## Overview

The Surviving Sepsis Campaign Hour-1 Bundle defines actions that must begin **within 1 hour of sepsis recognition**. Early bundle compliance reduces mortality by up to 40%.

**Time is tissue. Every hour of delayed treatment increases mortality by 7%.**

## The 5 Bundle Elements

### 1. Measure Lactate
- Draw serum lactate immediately
- **Re-measure lactate if initial >2 mmol/L** to assess response to resuscitation
- Lactate ≥4 mmol/L = septic shock even without hypotension → initiate vasopressors

### 2. Blood Cultures Before Antibiotics
- Obtain **at least 2 sets** (aerobic + anaerobic) before starting antibiotics
- Do NOT delay antibiotics >45 minutes to obtain cultures
- Include cultures from any indwelling devices if infection source suspected

### 3. Broad-Spectrum Antibiotics
- Administer within **1 hour of sepsis recognition**
- Choose empiric coverage based on suspected source:
  - **Unknown source / septic shock:** Vancomycin + Pip-Tazo or Meropenem
  - **Pneumonia (CAP):** Ceftriaxone + Azithromycin
  - **Abdominal source:** Pip-Tazo or Meropenem + Metronidazole
  - **Urinary source:** Ceftriaxone or Pip-Tazo
- De-escalate antibiotics at 48–72h based on culture results

### 4. IV Fluid Resuscitation
- **30 mL/kg crystalloid (normal saline or lactated Ringer's)** within 3 hours for hypoperfusion
- Reassess fluid responsiveness after each bolus — stop if signs of fluid overload
- Use dynamic measures (PPV, SVV, passive leg raise) to guide further resuscitation

### 5. Vasopressors for Persistent Hypotension
- Start vasopressors if MAP <65 mmHg despite adequate fluid resuscitation
- **First-line:** Norepinephrine (NE) — target MAP ≥65 mmHg
- **Add vasopressin 0.03 units/min** if NE dose >0.25 mcg/kg/min
- Place arterial line as soon as feasible for continuous blood pressure monitoring

## Escalation Criteria (Immediate Actions Required)

| Finding | Action |
|---|---|
| Lactate ≥4 mmol/L | Alert attending + ICU team, initiate vasopressors |
| SBP <90 or MAP <65 mmHg | IV fluid bolus NOW + vasopressors if no response |
| SpO2 <90% | Supplemental O2, consider intubation if SpO2 <88% |
| RR >25 or severe respiratory distress | Senior physician + respiratory therapy immediately |
| Altered mental status (GCS <14) | Neurology consult + urgent CT if new onset |
| No antibiotic given within 45 min | Escalate to attending — document reason for delay |

## Documentation Requirements

Every sepsis alert must document:
1. Time of sepsis recognition
2. Time of first antibiotic dose
3. Time of first fluid bolus
4. Lactate result and time drawn
5. Blood culture times and collection sites
6. Vasopressor initiation if applicable
7. Attending physician notification time
